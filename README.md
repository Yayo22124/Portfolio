## Ejecutar Localmente

Clonar proyecto

```bash
  git clone https://github.com/Yayo22124/Portfolio.git
```

Moverse al proyecto

```bash
  cd Portfolio
```

Instalar dependencias

```bash
  npm install
```

Ejecutar modo desarrollo

```bash
  npm run dev
```
